# Portfolio Architecte Sophie Bluel – JavaScript & API

---

## Contexte
Portfolio Architecte Sophie Bluel est un projet réalisé dans le cadre de la formation Intégrateur Web OpenClassrooms..  
L’objectif était de dynamiser un site portfolio existant en JavaScript, en interagissant avec une API REST, et en mettant en place une interface d’administration sécurisée..

---

## Objectifs pédagogiques
- Manipuler une API REST en JavaScript
- Gérer l’authentification utilisateur
- Créer une interface d’administration dynamique
- Mettre à jour le DOM à partir de données externes
- Structurer un code JavaScript clair et maintenable

---

## Technologies utilisées
- HTML5
- CSS3
- JavaScript (Vanilla)
- API REST

---

## Fonctionnalités principales
- Affichage dynamique des projets depuis l’API
- Filtrage des travaux par catégories
- Interface de connexion administrateur
- Ajout et suppression de projets en temps réel
- Mise à jour automatique de la galerie après modification

---

## Contraintes du projet
- JavaScript vanilla uniquement
- Communication avec une API existante
- Gestion sécurisée de l’authentification
- Respect du HTML et CSS fournis
- Compatibilité avec les navigateurs modernes

---

## Choix techniques
- Utilisation de `fetch` pour les appels API
- Stockage du token en session pour sécuriser la session administrateur
- Séparation claire des responsabilités dans les scripts JavaScript
- Manipulation dynamique du DOM basée sur les réponses de l’API

---

## Installation et lancement
Aucune installation particulière n’est nécessaire..

1. Cloner le dépôt
2. Lancer le serveur backend fourni
3. Ouvrir le fichier `index.html` dans un navigateur

---

## Points forts
- Interaction complète avec une API REST
- Authentification fonctionnelle
- Interface admin claire et réactive
- JavaScript structuré et lisible
- Mise à jour dynamique sans rechargement de page
